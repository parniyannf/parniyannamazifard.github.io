# StudySpot - Campus Study Space Finder

## Overview
Developed a real-time web application that displays live campus study space availability using color-coded occupancy indicators and user-submitted updates. Built as part of the WiE x ESSS Hackathon.

## Key Features
- **Real-time occupancy tracking** - Live updates on study space availability
- **Color-coded indicators** - Visual system for quick availability assessment
  - Green: Available
  - Yellow: Partially occupied
  - Red: Full
- **User-submitted updates** - Community-driven data collection
- **Responsive design** - Works on mobile and desktop devices
- **Campus map integration** - Visual location of study spaces

## Development Process

### Hackathon Challenge
- **Time constraint:** 12 hours
- **Team size:** 4 members
- **Goal:** Create a working prototype

### My Contributions
- Focused on responsive interface design
- Implemented real-time data integration
- Developed color-coded occupancy system
- Created user-friendly data submission form

## Technical Stack
- **Frontend:** HTML, CSS, JavaScript
- **Real-time updates:** WebSocket or polling mechanism
- **Responsive design:** Mobile-first approach
- **Deployment:** Web hosting platform

## Results
Successfully built and demonstrated a working prototype within the 12-hour time limit, receiving positive feedback from judges and users.

## Future Improvements
- Integration with campus WiFi data for automatic occupancy detection
- User authentication for reliable updates
- Historical data analysis for occupancy patterns
- Push notifications for favorite spaces
